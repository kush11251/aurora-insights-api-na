# Aurora Insights API
A high-performance analytics platform for data-driven decision making.
## Installation
1. Clone the repository: `git clone https://github.com/username/aurora-insights-api.git`
2. Initialize the Go module: `go mod init aurora-insights-api`
3. Install dependencies: `go get -u ./...`
## Run
To start the server, run: `go run src/main.go`
